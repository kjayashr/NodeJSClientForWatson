Visual Recognition project 

Welcome to the Visual Recognition project using Artificial Intelligence.
This on going research project at the Department of Software Engineering,makes use of Artificial Intelligence to determine anomolies of the human eye.


This project aims at responding users with immedieate scores of the uploaded eye image which is an healthiness index and helps to detect immediate anomalies.

Artificial Intelligence at ASU

Human Errors are most common in Medication and we at Department of Softare Engineering ASU Polytechnic believe that using Artificial Intelligence, We can solve more repetative tasks which involves human intelligence.

How do we do it ? 
We classify images of positive and negative samples of the human eye. 
After classification we train our classifier with these sample images. Any new incoming imges will be queried against this classifier to obtain the response. 
This response obtained by Artificial Intelligence reduces human errors in detecting eye anomolies.

TECHNOLOGY OVERVIEW

IBM Wastson Visual Recognition Service

Visual recognition service using IBM Watson. IBM Bluemix has IBM Watson Series of AI service. One such service is Visual Recognition Service. IBM Watson implements proprietary machine learning algorithms to perform learning.

Client : Node + Express + MongoDB

The user facient client application is being built using Node JS with Express framework. The application uses EJS templating engine and uses Mongo DB database

Application for Training Classifiers

The training of the custom classifier is done using a different application. The open source client for IBM Watson image recognition services integrated with various training APIs is implemented to build custom classifier.

CloudFounary and Containers

For the better performance cloudfoundary is used to push the code to IBM Bluemix environment on the go.
Containerization of the whole application is being researched on, to find the feasibility of hosting the application with minimum cost.
